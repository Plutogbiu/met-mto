# 运维内部工单系统设计文档

本目录包含煤矿运维内部工单系统的产品与技术设计材料。

当前项目定位为小团队内部使用，重点解决现场工单记录、到矿打卡、图片留痕、维保描述、日常巡检和报告生成问题。
打卡和工单导航计划接入腾讯地图 API，用于保存现场人员当前位置、维护煤矿坐标，并支持现场人员按工单导航到煤矿。

## 当前优先级

下一步建议先继续调整：

1. 数据库表结构：围绕用户、客户现场、工单、打卡、图片、巡检项、报告。
2. 报告模板：确定维保报告和巡检回执报告的展示格式。
3. 项目初始化：分别初始化 `mto-admin`、`mto-server`、`mto-app`。
